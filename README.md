# cqulug.club 网页源码

## 本地测试

首次使用先安装 jekyll 和相关软件包：

```bash
bundle install
```

之后使用 jekyll 来预览：

```bash
bundle exec jekyll server --watch
```

之后就可以通过访问 <http://127.0.0.1:4000/>（默认设置）来预览。
